Hedadings

The headings are a font from google called Anton set to standard font sizes with added letter spacing of .15em. I chose Anton for the thick lines to add contrast and help the user easily find the sections of the page.

Paragraph

The paragaphs and other type on the page are a font from google called Varela Round. I chose it for it's clean look and how easy it is to read.
